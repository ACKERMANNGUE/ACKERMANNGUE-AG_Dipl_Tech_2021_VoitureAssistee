71 // The 10-bit OCF (Opcode Command Field) of the HCI Opcode is further
72 // divided into two subsections:
73 // - Subgroup (3 bits):
74 // - 0 (LL)
75 // - 1 (L2CAP)
76 // - 2 (ATT)
77 // - 3 (GATT)
78 // - 4 (GAP)
79 // - 5 (UTIL)
80 // - 6 (Reserved)
81 // - 7 (User Profile)
82 // - Command (7 bits) or Profile (7 bits) if Subgroup value is set to
83 // User Profile (i.e., all ones) in which case, another octet is

290 // HCI extension events must start from 0x0400. The upper 6 bits of all
291 // zeros is reserved for the HCI extension embedded commands. The rest of
292 // the event bits should follow the HCI extension command format (i.e.,
293 // 3-bit Subgroup + 7-bit Event).
294 //
