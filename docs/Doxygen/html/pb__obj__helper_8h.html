mp_int_t pb_obj_get_hue ( mp_obj_t  arg)

Gets a HSV colorspace hue value.

If arg cannot be converted to an integer, an exception is raised. If arg is outside of the range of 0 to 359, it is treated as an angle and is reduced to the matching angle between 0 and 359.

Parameters
arg[in] A MicroPython object
Returns
An integer in the range of 0 to 359.
