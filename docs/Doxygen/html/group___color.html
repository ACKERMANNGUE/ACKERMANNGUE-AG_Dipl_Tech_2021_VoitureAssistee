void pbio_color_hsv_to_rgb ( const pbio_color_hsv_t *  hsv,
pbio_color_rgb_t *  rgb 
)

Converts HSV to RGB color value.

This method takes into account apparent brightness which works nicely with things that emit light, like LEDs.

This is not the direct inverse of pbio_color_rgb_to_hsv().

Parameters
[in]hsvThe source HSV color value.
[out]rgbThe destination RGB color value.
