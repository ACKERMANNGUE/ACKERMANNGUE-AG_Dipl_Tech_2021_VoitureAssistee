Error code.

Enumerator
PBIO_SUCCESS 

No error

PBIO_ERROR_FAILED 

Unspecified error (used when no other error code fits)

PBIO_ERROR_INVALID_ARG 

Invalid argument (other than port)

PBIO_ERROR_INVALID_PORT 

Invalid port identifier (special case of PBIO_ERROR_INVALID_ARG)

PBIO_ERROR_IO 

General I/O error

PBIO_ERROR_NO_DEV 

Device is not connected

PBIO_ERROR_NOT_IMPLEMENTED 

Feature is not yet implemented

PBIO_ERROR_NOT_SUPPORTED 

Feature is not supported on this device

PBIO_ERROR_AGAIN 

Function should be called again later

PBIO_ERROR_INVALID_OP 

Operation is not permitted in the current state

PBIO_ERROR_TIMEDOUT 

The operation has timed out

PBIO_ERROR_CANCELED 

The operation was canceled
