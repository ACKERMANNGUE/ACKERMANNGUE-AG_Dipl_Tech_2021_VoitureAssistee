Detailed Description

This file provides the high layer firmware functions to manage the following functionalities of the USB CDC_ECM Class:

Author
MCD Application Team
  • Initialization and Configuration of high and low layer
  • Enumeration as CDC_ECM Device (and enumeration for each implemented memory interface)
  • OUT/IN data transfer
  • Command IN transfer (class requests management)
  • Error management
