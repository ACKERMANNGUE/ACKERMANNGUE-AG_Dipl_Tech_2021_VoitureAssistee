◆ pbdrv_deinit_busy

#define pbdrv_deinit_busy ( )    (pbdrv_deinit_busy_count)

Tests if driver deinitialization is still busy.

After calling pbdrv_deinit(), the Contiki event loop should run until this returns true to wait for all drivers to be deinitalized.

◆ pbdrv_deinit_busy_down

#define pbdrv_deinit_busy_down ( )    (pbdrv_deinit_busy_count--)

Decreases driver deinit reference count.

Driver that have async deinit must call this when deinitialization is finished to indicate that deinit is no longer busy.
